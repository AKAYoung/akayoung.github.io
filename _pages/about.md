# Peisheng Yang

I am a Master's student in Software Engineering at Henan University. My research focuses on **Non-Intrusive Load Monitoring (NILM)**, **Smart Grid**, **Energy Internet**, and **time-series deep learning**.

My current work explores how synthetic data generation, cross-domain adaptation, and efficient Transformer architectures can improve appliance-level load perception under data-scarce, heterogeneous, and resource-constrained scenarios.

## Research Interests

- Non-Intrusive Load Monitoring
- Smart Grid and Energy Internet
- Time-Series Representation Learning
- Synthetic Load Data Generation
- Transformer and Diffusion Models
- Parameter-Efficient Fine-Tuning for Edge-Oriented NILM

## Current Research

My current research centers on **SynerBETA-NILM**, a two-stage NILM framework that combines synthetic load data generation with parameter-efficient cross-domain load disaggregation.

The framework consists of two main components:

- **SynerDiff**, a data augmentation model that integrates autoregressive Transformer modeling and diffusion-based generation for appliance-level power signals.
- **BETA-NILM**, a Transformer-based load disaggregation model with adapter-based parameter-efficient tuning and efficient attention.

## Selected Projects

- **SynerBETA-NILM**: A two-stage framework for synthetic load generation and parameter-efficient cross-domain NILM.
- **SynerDiff**: A synthetic appliance-level load data generation model based on Transformer and diffusion modeling.
- **BETA-NILM**: An adapter-based parameter-efficient Transformer model for cross-domain load disaggregation.
- **Multi-Scenario Power Load Perception System**: A practical load perception project involving data governance, sample library construction, feature design, and model training.
